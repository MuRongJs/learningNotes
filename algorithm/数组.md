可以通过异或(xor)运算符的特性来解答：

    1、  x ^ x = 0;
    2、  0 ^ x = x;
    3、  x ^ y ^ x = x ^ x ^ y = y;

通过者三个性质可以来解答这个算法。
```$xslt
var singleNumber = function(nums) {
    return nums.reduce((x, y) => x ^ y);
}
```
